Guide du dev + 1res tâches Voir le code (GitHub) 🎓 Apprendre à créer un jeu

🌱 Comment la borne grandit

  1. Un jeu = un fichier HTML autonome dans engine/games/ (il marche même en double-clic, hors-ligne).
  2. On ajoute une entrée au catalogue (engine/catalog.js) → le jeu apparaît tout seul dans le cabinet, le classement et le cache hors-ligne.
  3. Si le jeu enseigne une méthode, on ajoute son étude de cas à l'Atelier. Et on vérifie avec npm run check.
